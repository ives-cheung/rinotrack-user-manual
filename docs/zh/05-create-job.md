# 5. 创建您的第一个作业

## 5.1 新建车辆参数

1. 进入**作业管理** → **新建作业**
2. 点击**添加车辆**
3. 填写以下信息：
   - **车辆名称** - 如"约翰迪尔1860"
   - **车型型号** - 从列表中选择或手动输入
   - **轮距** - 农机前后轮的距离（单位：米）
   - **轴距** - 前轮到后轮的距离（单位：米）
4. 点击**保存**

## 5.2 测量并输入农具参数

**农具幅宽** 是自动驾驶的关键参数：

1. 测量农具的工作幅宽（单位：米）
   - 例如：播种机宽度为3.5米
2. 进入**作业管理** → **农具设置**
3. 点击**新建农具**
4. 填写：
   - **农具名称** - 如"约翰迪尔播种机"
   - **工作幅宽** - 输入测量的宽度
   - **农具类型** - 选择播种、喷洒、旋耕等
5. 点击**保存**

## 5.3 建立农场和地块

1. 进入**农场管理**
2. 点击**新建农场**
   - 输入农场名称（如"中关村农场"）
   - 选择位置
   - 保存
3. 在农场下建立**地块**
   - 点击**添加地块**
   - 输入地块编号（如"A1地块"）
   - 标记地块位置
4. 创建**任务**
   - 点击**新建任务**
   - 输入任务名称（如"冬麦播种2026"）
   - 选择地块和农具
   - 设置任务日期
   - 保存
