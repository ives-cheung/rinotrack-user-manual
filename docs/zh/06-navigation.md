# 6. 创建导航线开始作业

## 6.1 创建路径

1. 进入**导航**界面
2. 选择您要作业的地块
3. 点击**创建新路径**
4. 屏幕上出现地块地图
5. **点击标记**地块的四个角落（或按实际形状标记）
6. 系统自动生成导航线
7. 点击**生成路径**确认

## 6.2 认识各种路径类型

系统支持多种导航路径类型：

- **平行线** - 最常用，按间距自动生成平行的导航线
- **环形** - 绕地块周围工作
- **对角线** - 适用于某些特殊地块
- **自定义** - 手动绘制导航线

## 6.3 启动自动驾驶

1. 路径创建完成后，点击**开始作业**
2. 系统进入自动驾驶模式
3. 农机将自动沿着规划线路行驶
4. 屏幕显示：
   - 当前位置
   - 与导航线的偏差
   - 实时速度
   - 作业进度

## 6.4 配置掉头和转向

在地块两端的掉头区域：

1. 系统会自动**减速提示**
2. 您可以选择：
   - **自动掉头** - 系统自动转向到下一条导航线
   - **手动掉头** - 您手动操作，系统辅助
3. 配置掉头的转角和宽度
4. 保存设置以便下次作业使用
