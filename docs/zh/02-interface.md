# 2. 认识主界面

## 2.1 顶部状态栏

主界面顶部显示系统当前的连接和信号状态：

| 图标 | 含义 | 说明 |
|------|------|------|
| 📶 信号强度 | GNSS信号 | 显示卫星信号质量，格数越多信号越好 |
| 📡 Wi-Fi | 无线网络 | 连接到本地无线网络的状态 |
| 📶 4G | 移动网络 | 4G信号强度指示 |
| 🔋 电量 | 系统电源 | 设备电池或电源状态 |
| ⚙️ 设置 | 系统状态 | 系统是否正常工作 |

## 2.2 中央操作区

屏幕中央显示农场地图和当前作业的导航信息：

- 当前位置标记
- 规划的导航线
- 已完成的作业区域
- 实时速度和位置信息
