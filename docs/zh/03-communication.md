# 3. 通信配置

## 3.1 差分接入方式

系统支持多种差分GNSS信号接入方式：

### 3.1.1 实时动态（RTK）定位
- 精度：±2-3厘米
- 适用于高精度作业
- 需要基站或差分信号源

### 3.1.2 DGPS方式
- 精度：±0.5-1米
- 覆盖范围广
- 成本相对较低

### 3.1.3 网络差分（NTRIP）
- 通过4G/Wi-Fi接收差分数据
- 无需自建基站
- 适合移动作业

## 3.2 配置步骤

1. 进入**设置** → **通信配置**
2. 选择您需要的差分接入方式
3. 输入基站信息或账号密码
4. 点击**连接**进行测试
5. 确认信号强度显示正常后保存
