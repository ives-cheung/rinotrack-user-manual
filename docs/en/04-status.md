# 4. System Status Check

## 4.1 System Status Panel

Enter the **Status Check** module to view real-time parameters of the system:

| Parameter | Description |
|-----------|-------------|
| GNSS Status | Satellite signal quality, typically displays number of satellites |
| Positioning Accuracy | Current positioning accuracy level |
| Steering Wheel Angle | Current angle of the electric steering wheel |
| Vehicle Speed | Real-time speed of the agricultural machinery |
| CPU Temperature | Controller operating temperature |
| Sensor Status | Operating status of each sensor |

## 4.2 Normal Operation Indication

If all system parameters display normally (green), **no adjustments are necessary**. The system will handle operations automatically.

> ⚠️ If red warnings or abnormal parameters appear, please refer to the troubleshooting section or contact technical support.
