# 1. System Hardware Components

## 1.1 Main Components

The autonomous agricultural machinery driving system consists of the following core components:

- **Display Screen** - Used to display navigation maps, system status, and operation interface
- **Electric Steering Wheel** - Implements automatic steering control
- **ECU Controller** - Central Processing Unit (a box-shaped device similar to GNSS receivers), responsible for the system's core calculations and control
- **Connection Cables** - Communication and power cables between components

## 1.2 Model Differences

Different machinery models may have different hardware configurations. **If you need to understand the specific configuration of your agricultural machinery model, please contact your local dealer for professional training**.

> ℹ️ This manual only introduces basic software functions and general operating methods.
