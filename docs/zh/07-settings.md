# 7. 更多设置

进入**设置**菜单可以自定义系统参数：

## 7.1 语言和单位

- **显示语言** - 选择中文、英文等
- **距离单位** - 米、公里、英里
- **面积单位** - 亩、公顷、英亩
- **温度单位** - 摄氏度、华氏度

## 7.2 时间和地区

- **时区** - 自动设置或手动调整
- **日期格式** - 选择偏好的日期显示方式
- **地区设置** - 用于本地化内容

## 7.3 演示模式

- 开启后可以在**没有实际GNSS信号**的情况下体验软件
- 适合在室内或新用户培训
- 不会保存真实的作业数据

## 7.4 功能开通

某些高级功能需要通过：

- **激活码** - 输入授权码开启功能
- **订阅服务** - 订阅云服务和数据同步
- **模块升级** - 购买并安装额外功能模块

## 7.5 其他常用设置

| 功能 | 说明 |
|------|------|
| 音量控制 | 调整语音提示的音量 |
| 亮度调节 | 适应不同光线条件 |
| 自动保存 | 定期保存作业数据 |
| 数据备份 | 备份农场和作业记录 |
| 日志导出 | 导出系统日志用于诊断 |
| 关于本软件 | 查看版本号和更新信息 |
