# 农机自动驾驶软件新手教程

👋 欢迎使用农机自动驾驶系统！

本手册为新用户提供完整的软件操作指南，帮助您快速上手自动驾驶功能。

## 📚 教程章节

| 章节 | 内容 |
|------|------|
| [1. 系统硬件组成](01-hardware.md) | 了解屏幕、电动方向盘、ECU和线缆等核心部件 |
| [2. 认识主界面](02-interface.md) | 学习各种图标含义和界面布局 |
| [3. 通信配置](03-communication.md) | 配置差分接入方式（RTK、DGPS、NTRIP） |
| [4. 状态检查](04-status.md) | 查看和理解系统各项参数 |
| [5. 创建作业](05-create-job.md) | 新建车辆、农具、农场和任务 |
| [6. 导航线和自动驾驶](06-navigation.md) | 创建导航线并启动自动驾驶 |
| [7. 更多设置](07-settings.md) | 语言、单位、演示模式等个性化设置 |

## 🚀 快速开始路线

1️⃣ 先从 **[系统硬件组成](01-hardware.md)** 了解整体设备  
2️⃣ 然后 **[认识主界面](02-interface.md)** 熟悉操作  
3️⃣ 配置 **[通信参数](03-communication.md)** 连接信号  
4️⃣ 创建 **[第一个作业](05-create-job.md)**  
5️⃣ 最后 **[启动自动驾驶](06-navigation.md)** 开始工作  

## 💡 小贴士

- 不同型号的农机硬件可能有所差异，如有问题请联系经销商
- 系统各项参数正常显示时无需调整，系统会自动处理
- 演示模式适合在室内或无信号环境下学习操作
