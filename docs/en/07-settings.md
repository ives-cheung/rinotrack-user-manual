# 7. More Settings

Go to the **Settings** menu to customize system parameters:

## 7.1 Language and Units

- **Display Language** - Choose Chinese, English, etc.
- **Distance Units** - Meters, kilometers, miles
- **Area Units** - Mu (Chinese acre), hectares, acres
- **Temperature Units** - Celsius, Fahrenheit

## 7.2 Time and Region

- **Time Zone** - Automatically set or manually adjust
- **Date Format** - Choose your preferred date display format
- **Region Settings** - Used for localizing content

## 7.3 Demo Mode

- When enabled, you can experience the software **without actual GNSS signals**
- Suitable for indoor learning or new user training
- Does not save actual job data

## 7.4 Feature Activation

Some advanced features require:

- **Activation Code** - Enter authorization code to enable features
- **Service Subscription** - Subscribe to cloud services and data synchronization
- **Module Upgrade** - Purchase and install additional feature modules

## 7.5 Other Common Settings

| Feature | Description |
|---------|-------------|
| Volume Control | Adjust the volume of voice notifications |
| Brightness Adjustment | Adapt to different lighting conditions |
| Auto Save | Periodically save job data |
| Data Backup | Backup farm and job records |
| Log Export | Export system logs for diagnostics |
| About This Software | View version number and update information |
