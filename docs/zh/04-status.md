# 4. 状态检查

## 4.1 系统状态面板

进入**状态检查**模块，可以查看系统的各项实时参数：

| 参数 | 说明 |
|------|------|
| GNSS状态 | 卫星信号质量，通常显示卫星数量 |
| 定位精度 | 当前的定位精度等级 |
| 方向盘角度 | 电动方向盘的当前转角 |
| 车速 | 农机的实时速度 |
| CPU温度 | 控制器的工作温度 |
| 传感器状态 | 各传感器工作状态 |

## 4.2 正常运行指示

如果系统各项参数显示正常（绿色），**则无需进行任何调整**。系统会自动处理各项工作。

> ⚠️ 如果出现红色警告或异常参数，请参考故障排查章节或联系技术支持。
