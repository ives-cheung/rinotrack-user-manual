# 6. Creating Navigation Lines and Starting Operations

## 6.1 Creating Paths

1. Go to **Navigation** interface
2. Select the field block you want to work on
3. Click **Create New Path**
4. A field block map appears on the screen
5. **Click to mark** the four corners of the field block (or mark according to actual shape)
6. The system automatically generates navigation lines
7. Click **Generate Path** to confirm

## 6.2 Understanding Different Path Types

The system supports multiple navigation path types:

- **Parallel Lines** - Most common, automatically generates parallel navigation lines at set intervals
- **Circular** - Work around the perimeter of the field
- **Diagonal** - Suitable for certain special field shapes
- **Custom** - Manually draw navigation lines

## 6.3 Starting Autonomous Driving

1. After path creation is complete, click **Start Job**
2. The system enters autonomous driving mode
3. The agricultural machinery will automatically drive along the planned route
4. The screen displays:
   - Current position
   - Deviation from the navigation line
   - Real-time speed
   - Job progress

## 6.4 Configuring Headland Turns and Steering

In the turning areas at both ends of the field:

1. The system will automatically **reduce speed and notify**
2. You can choose:
   - **Automatic Turn** - System automatically turns to the next navigation line
   - **Manual Turn** - You operate manually; system assists
3. Configure the turning angle and width
4. Save settings for use in future operations
